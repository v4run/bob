'bob' the builder
===

Build go projects automatically when files are modified. 

Getting
===

```
git clone git@github.com:v4run/bob.git
```

Usage
===


```
bob [options]
options:
	-d, -directory Directory   The directory to watch.
	-v, -version   Version     Prints the version.
	-h, -help      Help        Show this help.

```
